[[cleanup-steps]]
= Clean up steps
:numbered!:

[[cleanup-step1]]
== Step 1 -> Stop the containers

To stop all the containers, type:

[,console]
----
$ ./docker/down.sh
----

[[cleanup-step2]]
== Step 2 -> Clean up

To remove all the containers and images, type:

[,console]
----
$ ./docker/clean.sh
----

[WARNING]
.Warnings:
====
. The script `clean.sh` will destroy any data generated by these containers.
====
